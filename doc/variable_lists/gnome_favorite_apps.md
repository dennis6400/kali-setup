# [gnome_favorite_apps](gnome_favorite_apps.md)

## Syntax

```yml
gnome_favorite_apps: "['<App 1>', '<App 2>', ...]"
```

## Example

```yml
gnome_favorite_apps: "['org.gnome.Nautilus.desktop', 'firefox-esr.desktop', 'chromium.desktop', 'org.gnome.Terminal.desktop', 'terminator.desktop', 'org.gnome.TextEditor.desktop', 'sublime_text.desktop', 'sublime_merge.desktop', 'obsidian.desktop', 'cherrytree.desktop', 'kali-msfconsole.desktop', 'kali-burpsuite.desktop', 'hydra-gtk.desktop', 'org.wireshark.Wireshark.desktop', 'htop.desktop', 'org.gnome.SystemMonitor.desktop', 'org.gnome.tweaks.desktop', 'org.gnome.Settings.desktop']"
```

## Default Value

```yml
empty
```
