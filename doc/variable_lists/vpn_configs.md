# [vpn_configs](vpn_configs.md)

## Syntax

```yml
vpn_configs:
  - { dir: "<source and destination dir>", file: "<vpn file>" }
```

## Example

```yml
vpn_configs:
  - { dir: "example", file: "example.ovpn" }
```

## Default Value

```yml
empty
```
